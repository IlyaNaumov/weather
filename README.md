Установка:
1.Перенести содержимое папки в удобную для вас директорию
2.Открыть файл "task_schedule_forecast.xml" и указать текущие пути к папке "forecast36"
Если папка сохранена в корневую директорию диска "C"- пропустить п.2:
                                   <Command>C:\forecast36\Scripts\python.exe</Command>
                                   <Arguments>C:\\forecast36\forecast36.py</Arguments>
                                   <WorkingDirectory>C:\\forecast36\</WorkingDirectory>
3.Далее нужно открыть планировщик задач (taskchd.msc) и выполнить импорт задачи, для этого на панели действий:
 Действие>импортировать задачу...>"task_shedule_forecast.xml"
4.В папке forecast36 открыть settings.cfg:
   4.1 В строке "appid =" указать APPKEY полученый при регистрации на OpenWeatherMap;
   4.2 В строке "Path =" изменить текущий путь к папке forecast36, 
   если папка сохранена в корневую директорию диска "C" - пропустить п.4.2

Запуск скрипта будет осуществляться по расписанию планировщика задач после включения рабочей машины каждые 3 часа, 
все файлы будут сохраняться в папке "C:\forecast36\weather_archive\" в формате .csv
